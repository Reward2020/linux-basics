# Linux Tips & Cheatsheet

Цей репозиторій містить корисні підказки, команди та короткі довідки для ефективної роботи з Linux через термінал.

## Що тут є?

- Основні команди Linux (ls, cd, cp, mv, cat, pwd, та інші)
- Пояснення вбудованих та зовнішніх команд
- Приклади використання команд з опціями
- Підсумкові таблиці для швидкого орієнтування
- Поради щодо роботи з процесами (`bg`, `fg`, `nohup`, `tmux`/`screen`) та іншими інструментами

## Для кого цей репозиторій?

- Для початківців, які хочуть швидко освоїти базові команди Linux
- Для тих, хто хоче мати під рукою шпаргалку по часто використовуваних командам
- Для тих, хто прагне краще розуміти роботу оболонки bash та терміналу

## Як користуватись?

Просто переглядайте файли, шукайте потрібні команди і приклади. Рекомендується клонувати репозиторій або зберегти його у зручному місці для швидкого доступу.
